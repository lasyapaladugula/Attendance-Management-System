# Student Attendance Management System

A web-based Student Attendance Management System developed using HTML, CSS, JavaScript, Node.js, Express, and MongoDB. This application helps institutions manage and track student attendance efficiently.

## Features
- Admin login to manage attendance
- Student login to view attendance dashboard
- Subject-wise attendance tracking
- Attendance percentage calculation
- Date-wise subject filtering using calendar
- Clean and responsive dashboard UI

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## Project Modules
1. Admin Module
   - Mark attendance
   - Manage student records

2. Student Module
   - Login authentication
   - View attendance dashboard
   - View subject-wise attendance
   - Date-based attendance view
